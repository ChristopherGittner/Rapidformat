Rapidformat
===========

This Software will format .mod or .sys Files

Currently it supports:
- Indentation
- Keyword Writing e.g. movej -> MoveJ

PREREQUISITES
=============
- CMAKE ( Version > 3.0 for Windows )
- wxWidgets > 3.0
- Boost

BUILDING
========

Linux:

    mkdir build
    cd build
    cmake ..
    make

Windows:

    On Windows use the cmake-gui
